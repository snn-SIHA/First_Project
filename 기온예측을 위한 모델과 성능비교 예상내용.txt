<모델과 성능비교 예상내용>

회귀모델
linearRegression : RMSE(평균 제곱근 오차)나 R²(결정 계수)와 같은 지표를 사용할 수 있습니다
선형성 가정: 선형 회귀는 독립 변수와 종속 변수 간의 선형 관계를 가정합니다. 기온 데이터가 비선형적인 패턴을 보일 경우, 선형 회귀는 적합하지 않을 수 있습니다.
이상치에 민감함: 선형 회귀는 이상치(outlier)에 민감하여, 이상치가 모델의 성능에 큰 영향을 미칠 수 있습니다.

비교대상(다른 모델):
svr : svc와 비슷하지만 이진분류가 아니라 연속적인 값을 예측. 단, 하이퍼파라미터 조정이 필요하고 이를 위해 교차 검증 등의 기법을 사용할 수 있습니다. 
대규모 데이터셋에 대해 계산 비용이 높을 수 있습니다. 특히, 데이터 포인트가 많을 경우 학습 시간이 길어질 수 있습니다.
decisionTree : 과적합(overfit)이 있을 수 있어서 방지 필요
RandomForest : decisionTree보다 과적합이 안 일어남. 


결론:
선형 회귀는 기온 예측에 사용할 수 있는 기본적인 모델이며, 데이터가 선형적인 경향을 보일 경우 유용합니다. 
그러나 기온 데이터가 비선형적인 패턴을 보일 가능성이 높기 때문에, 다른 비선형 회귀 모델(예: 결정 트리, SVR 등)과 비교하여 성능을 평가하는 것이 좋습니다. 
선형 회귀는 기온 예측의 기초 모델로 시작한 후, 필요에 따라 더 복잡한 모델로 확장하는 접근 방식이 유용할 수 있습니다.


<성능평가>

혼동 행렬(Confusion Matrix)은 주로 이진 분류 문제에서 모델의 성능을 평가하는 데 사용되는 도구입니다. 혼동 행렬은 실제 클래스와 예측 클래스 간의 관계를 나타내며, True Positive, True Negative, False Positive, False Negative의 수치를 제공합니다. 이를 통해 정확도, 정밀도, 재현율, F1 점수 등의 성능 지표를 계산할 수 있습니다.

그러나 선형 회귀(Linear Regression), SVR(Support Vector Regression), 결정 트리(Decision Tree), 랜덤 포레스트(Random Forest)는 모두 회귀 모델입니다. 이 모델들은 연속적인 값을 예측하는 데 사용되며, 혼동 행렬을 직접적으로 사용할 수 없습니다. 혼동 행렬은 이진 분류 문제에 적합하기 때문입니다.

## 회귀 모델의 성능 비교 방법

회귀 모델의 성능을 비교하기 위해서는 다음과 같은 지표를 사용할 수 있습니다:

1. RMSE (Root Mean Squared Error): 예측값과 실제값 간의 차이를 제곱하여 평균을 구한 후, 제곱근을 취한 값입니다. 값이 작을수록 모델의 성능이 좋습니다.

2. MAE (Mean Absolute Error): 예측값과 실제값 간의 절대 차이의 평균입니다. 값이 작을수록 모델의 성능이 좋습니다.

3. R² (Coefficient of Determination): 모델이 데이터의 변동성을 얼마나 설명하는지를 나타내는 지표입니다. 1에 가까울수록 모델의 설명력이 높습니다.

4. MSE (Mean Squared Error): 예측값과 실제값 간의 차이를 제곱하여 평균을 구한 값입니다. 값이 작을수록 모델의 성능이 좋습니다.

### 성능 비교 방법

1. 모델 훈련: SVR, 결정 트리, 선형 회귀, 랜덤 포레스트 모델을 각각 훈련합니다.

2. 예측 수행: 각 모델을 사용하여 테스트 데이터에 대한 예측을 수행합니다.

3. 성능 지표 계산: 각 모델의 예측 결과를 바탕으로 RMSE, MAE, R² 등의 성능 지표를 계산합니다.

4. 비교: 계산된 성능 지표를 비교하여 각 모델의 성능을 평가합니다.

## 결론

RMSE, MAE, R² 등의 지표를 사용하여 선형회귀, SVR, 결정 트리, 랜덤 포레스트 모델의 성능을 비교할 수 있습니다. 각 모델의 예측 성능을 평가하고, 최적의 모델을 선택하는 데 이러한 지표를 활용하는 것이 좋습니다.
